# 056. Harbor 이미지 목록과 태그 목록을 좌우 2단 패널로 재배치

- 일시: 2026-05-08
- 사용자 요청:
  - "그럼 이제 이미지 저장소를 눌렀을 때 나오는 하단의 태그 목록을 그냥 이미지 저장소 리스트 오른쪽으로 위치를 이동해줘"

## 처리 내용

1. Harbor 프로젝트 상세 영역에서 `이미지 목록(repository list)`과 `태그 목록(tag list)`을 별도 패널로 분리했다.
2. 기존 하단 배치 대신 `좌측: 이미지 목록`, `우측: 태그 목록` 2단 레이아웃으로 재배치했다.
3. 좁은 화면에서는 기존처럼 위아래로 자연스럽게 쌓이고, 넓은 화면에서는 오른쪽 패널에 태그가 보이도록 반응형 grid를 적용했다.
4. Harbor 프로젝트 삭제, 이미지 선택 삭제, 태그 선택 삭제, 태그 검색, 태그 로딩 상태 표시는 그대로 유지했다.
