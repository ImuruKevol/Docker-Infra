# 로그인 화면 운영 문구와 키워드 카드 개선

- **ID**: 001
- **날짜**: 2026-06-15
- **유형**: 버그 수정

## 작업 요약
로그인 화면의 오래된 운영 문구를 현재 Docker Infra 운영 흐름에 맞게 정리했습니다.
서버 상태, 서비스 배포, DDNS/SSL, 작업 추적 중심의 키워드 카드로 바꾸고 로그인 영역 문구도 운영 콘솔 기준으로 다듬었습니다.

## 원문 요청사항
```text
작업 시작

로그인 화면에 있는 키워드들이나 설명이 너무 옛날에 만들어서 현 상태와 맞지 않아.
실제 운영을 한다 했을 때 어울리도록 문구와 키워드 카드 등을 알맞게 개선해줘.
```

## 변경 파일 목록
- `src/app/page.access/view.pug`
  - 히어로 제목과 설명을 실제 운영 점검/조치 흐름에 맞게 수정했습니다.
  - 키워드 카드를 서버 상태, 서비스 배포, DDNS/SSL, 작업 추적으로 재구성했습니다.
  - 로그인 패널의 접속 문구를 운영자 로그인 기준으로 정리했습니다.
- `devlog.md`
  - 작업 요약 행을 추가했습니다.
