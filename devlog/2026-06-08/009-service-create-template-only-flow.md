## 작업 내용

- 서비스 생성 화면에서 AI/직접 작성 생성 옵션과 단계 사이드바를 제거하고 템플릿 기반 단일 화면으로 재구성했다.
- 기존 3단계 도메인 설정을 템플릿 입력 화면으로 이동해 서비스 이름, 템플릿 변수, 도메인 설정을 한 번에 입력하도록 했다.
- 생성 버튼 클릭 시 템플릿 초안을 적용한 뒤 자동 점검 모달을 띄우고, 점검 통과 시 생성 진행/완료 상태를 표시한 뒤 모달을 자동으로 닫고 목록으로 이동하도록 했다.
- 점검 실패 시 같은 모달에 preflight 항목과 조치 문구를 표시하고 돌아가기 버튼으로 입력 화면에 복귀하도록 했다.
