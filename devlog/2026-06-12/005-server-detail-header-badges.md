## 작업 요약
서버 관리 상세 화면 헤더에서 불필요한 "서버 상세" 뱃지를 제거했습니다.
서버 역할, IP, 정보 수집 상태가 같은 뱃지 그룹 안에서 `역할 -> IP -> 정보 수집 중` 순서로 표시되도록 정리했습니다.

```text
- "서버 상세" 뱃지 삭제
- "정보 수집 중" 뱃지 위치를 "일반 서버" 뱃지 오른쪽으로 이동
- IP와 id값같은걸 표시하고 있는데 id값같은건 제거, IP는 뱃지 형식으로 수정 후 정보 수집 중 뱃지 왼쪽으로 이동
```
