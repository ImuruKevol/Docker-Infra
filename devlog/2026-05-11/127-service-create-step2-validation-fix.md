# 127. 서비스 생성 AI 초안 후 2단계 진행 차단 수정

- 날짜: 2026-05-11
- 요청: "서비스 생성 시 AI로 초안 생성 후 2단계에서 넘어가지질 않아. 확인해줘."

## 작업 내용

AI 초안 적용 후 2단계 검증이 저장/렌더 규칙보다 엄격해 다음 단계로 진행하지 못할 수 있는 조건을 정리했다.

- 이미지 태그는 비어 있으면 backend에서 `latest`로 처리하므로, 2단계 검증에서는 이미지 이름만 필수로 확인하도록 변경했다.
- 공개 포트가 없는 내부 서비스도 도메인을 사용하지 않는 서비스로 계속 진행할 수 있도록 2단계의 공개 포트 필수 조건을 제거했다.
- 도메인 연결에 필요한 포트 검증은 실제 도메인을 선택하는 3단계로 이동했다.
- 공개 포트가 없으면 등록 도메인 선택 버튼을 비활성화하고, 포트를 추가해야 한다는 안내를 표시하도록 했다.
- 고급 설정에서 포트를 추가/삭제할 때 즉시 화면 상태가 갱신되도록 `service.render()`를 호출했다.
